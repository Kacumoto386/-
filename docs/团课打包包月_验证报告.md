## 自动编号
- 打包产品编号: `GP + 日期 + 4位序号`（如 GP202605070001）
- 包月团课编号: `MP + 日期 + 4位序号`（如 MP202605070002）
- 售出打包生成课程包编号: `PK + 日期 + 4位序号`

## 新增Excel Sheet
- **团课打包产品**（12列）：打包编号、打包名称、包含课程、课程名称列表、打包类型、总次数、标准售价、优惠售价、有效期(天)、状态、创建日期、备注
- **包月团课**（13列）：月卡编号、月卡名称、会员编号、会员姓名、包含课程、课程名称列表、售价、有效期起、有效期止、状态、购买日期、备注、门店编号
