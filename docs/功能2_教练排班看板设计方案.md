# 功能2：教练排班看板 设计方案

## 需求分析
预约按日期查，没有教练维度的时间视图。需要一个「教练日程看板」来按周展示教练的预约分布。

## 设计思路

### 不新增Sheet
基于现有的"预约管理"Sheet数据，做视图层面的展示。

### 视图设计

**布局**：周视图日历，横向=教练，纵向=时间段

```
            教练A    教练B    教练C    ...
08:00      已预约    空闲     已签到
09:00      空闲     已完成   已预约
10:00      已签到    空闲     空闲
...
```

### 业务层设计（core/business.py）

新增方法：
- `get_coach_schedule(week_start)` - 获取指定周的教练排班数据
- `get_all_trainers()` - 获取所有教练列表（从员工表筛选）
- `get_weekly_bookings(week_start)` - 获取一周的预约数据

### GUI设计

**新增界面**：`gui/schedule_frame.py`
- 放在"统计分析"分组下
- 功能：
  - 周导航（上一周/下一周/本周）
  - 按周显示教练排班
  - 时间段 x 教练的矩阵视图
  - 颜色标识预约状态（已预约=蓝色，已签到=绿色，已完成=灰色，已取消=红色）

## 数据来源
- 预约管理 Sheet → 按教练 + 日期筛选
- 员工信息 Sheet → 获取教练列表
